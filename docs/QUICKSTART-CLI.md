# Sphere CLI

The Sphere CLI is a **standalone package** — it is no longer part of this SDK repo, and its full
command reference lives with it (kept in sync there, not here):

- **Package:** [`@unicity-sphere/cli`](https://github.com/unicity-sphere/sphere-cli)
- **Install:** `npm install -g @unicity-sphere/cli`
- **Use:** `sphere --help`

> The old `npm run cli` script in `sphere-sdk` no longer works. See the CLI repo above for the
> complete, up-to-date command reference (wallet, payments, messaging, invoicing).
